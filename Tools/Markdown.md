## 一、基础语法

### 1、标题

| Markdown |  HTML  |
| :------: | :----: |
|   `#`    | `<h1>` |
|   `##`   | `<h2>` |
|   ...    |  ...   |
| `######` | `<h6>` |

### 2、段落

|  Markdown  | HTML  |
| :--------: | :---: |
| 空白行分隔 | `<p>` |

### 3、换行

|     Markdown      |  HTML  |
| :---------------: | :----: |
| 多个空格+回车换行 | `<br>` |

### 4、粗体和斜体

**粗体**

|        Markdown        |    HTML    |
| :--------------------: | :--------: |
| 内容前后加两个`*`或`_` | `<strong>` |

**斜体**

|        Markdown        |  HTML  |
| :--------------------: | :----: |
| 内容前后加一个`*`或`_` | `<em>` |

**粗体和斜体**

|        Markdown        |             HTML             |
| :--------------------: | :--------------------------: |
| 内容前后加三个`*`或`_` | `<strong><em></em></strong>` |
